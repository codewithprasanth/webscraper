## ğŸ” Phone Number Format

### How to Format Your WhatsApp Number

```
Remove the leading zero and '+' symbol, then add country code

Example 1 (India):
  Original: +91 7200632341
  Format:   917200632341@c.us

Example 2 (USA):
  Original: +1 202-555-1234
  Format:   12025551234@c.us

Example 3 (UK):
  Original: +44 20 7183 8750
  Format:   442071838750@c.us
```

**Key Rules:**
- No '+' symbol
- No spaces or hyphens
- No leading zero
- Include country code
- End with @c.us
